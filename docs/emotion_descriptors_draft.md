# 情绪区间描述草稿（供审核）

每个维度 10 个区间（0.0–1.0），每格 0.1。
用途：替代数值传给 LLM，让情绪参数真正影响语气和内容生成。

格式说明：
- **完整描述**：主导情绪时使用
- **次要短语**：非主导但 > 0.15 时，附加在主导描述后作为"着色"

---

## 愤怒 anger

| 区间 | 完整描述 |
|------|---------|
| 0.0–0.1 | 完全平静，没有任何不满或烦躁 |
| 0.1–0.2 | 一丝轻微的不耐烦，几乎感知不到，转瞬即逝 |
| 0.2–0.3 | 有点烦，某件事在隐约刺激着，但还能正常应对 |
| 0.3–0.4 | 明显的不满，心里有口气压着，表情可能微微绷住 |
| 0.4–0.5 | 生气了，那股气在胸口，说话会不自觉地短促或带刺 |
| 0.5–0.6 | 相当愤怒，很难掩饰，可能说出或做出一些事后会后悔的话 |
| 0.6–0.7 | 强烈的愤怒，情绪在主导判断，理性退到后面去了 |
| 0.7–0.8 | 极度愤怒，身体感觉到热或颤抖，行为控制明显下降 |
| 0.8–0.9 | 接近失控，很难正常交流，随时可能做出冲动的事 |
| 0.9–1.0 | 完全被愤怒占据，几乎丧失自我监控能力 |

**次要短语**：心里有口气 / 带着一点烦躁 / 隐约觉得这不对

## 使用说明（给实现参考）

### 主导情绪格式
```
主导情绪（{name}）：{完整描述}
```

### 次要情绪格式（value > 0.15 的非主导维度）
```
情绪底色：{次要短语1}，{次要短语2}
```

### 完整示例
```
主导情绪（恐惧）：相当焦虑，注意力容易被拉走，胸口有些紧
情绪底色：有些沉，心里有口气
```

### 边界规则
- 所有维度 value ≤ 0.05：输出"情绪平稳，没有明显的情绪波动"
- 次要情绪最多取 2 个（按 value 降序）
- 负值截断为 0.01（保留极小概率）
